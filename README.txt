KoF Combo Hitbox Viewer, version 1.0.2
https://github.com/odabugs/kof-combo-hitboxes

Basic usage instructions:
1) Start desired game
2) Start kof-hitboxes.exe
3) Additional usage instructions will be presented in the console window

Windows Vista or newer is required, with Windows Aero and DWM enabled.
PS2 games are tested with PCSX2 1.4.0.
Configuration settings can be changed in default.ini.
Embedded LuaJIT is version 2.0.5, compiled for 32-bit (http://luajit.org/).

Supported games:
- Steam:
  - King of Fighters '98 Ultimate Match Final Edition (Steam and GOG.com releases both supported)
  - King of Fighters 2002 Unlimited Match (Steam and GOG.com releases both supported)
  - Guilty Gear XX #Reload (NOTE: Non-Steam releases of this game are not tested)
  - Guilty Gear XX Accent Core +R
- PlayStation 2 (via PCSX2):
  - King of Fighters XI
  - King of Fighters '98 Ultimate Match
  - King of Fighters 2002 Unlimited Match (original and Tougeki Ver. releases both supported)
  - King of Fighters NeoWave
  - NeoGeo Battle Coliseum
  - Capcom vs. SNK 2 (currently only supports NTSC-U version)
  - Capcom Fighting Evolution (currently only supports NTSC-U version)
